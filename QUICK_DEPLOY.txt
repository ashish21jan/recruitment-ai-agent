STEP 1: PUSH TO GITHUB (5 minutes)
--------------------------------------------------------------------------------
In your terminal, run these commands:

    git init
    git add .
    git commit -m "Initial commit - Recruitment AI Agent"

Then:
1. Go to https://github.com/new
2. Create repository named: recruitment-ai-agent
3. Don't initialize with README
4. Copy the commands shown and run them, OR run:

    git remote add origin https://github.com/YOUR_USERNAME/recruitment-ai-agent.git
    git branch -M main
    git push -u origin main


STEP 2: DEPLOY TO RENDER (5 minutes)
--------------------------------------------------------------------------------
1. Go to https://render.com and sign up (use GitHub login)

2. Click "New +" → "Web Service"

3. Connect your GitHub repository: recruitment-ai-agent

4. Configure these settings:
   ┌─────────────────────────────────────────────────────────────┐
   │ Name:           recruitment-ai-agent                        │
   │ Region:         Oregon (or closest to you)                  │
   │ Branch:         main                                        │
   │ Runtime:        Python 3                                    │
   │ Build Command:  pip install -r requirements.txt             │
   │ Start Command:  uvicorn main:app --host 0.0.0.0 --port $PORT│
   │ Plan:           FREE ⭐                                      │
   └─────────────────────────────────────────────────────────────┘

5. Add Environment Variable:
   Key:   GEMINI_API_KEY
   Value: [Your actual Gemini API key]

6. Click "Create Web Service"

7. Wait 5-10 minutes for first deployment


STEP 3: GET YOUR URL
--------------------------------------------------------------------------------
Once deployed, your URL will be:

    https://recruitment-ai-agent-XXXX.onrender.com

Copy this URL and share it with anyone!
